 University View Folder Contains the following:
1) University.swift - Contains the structure and the array that holds the infomation.\
2) Universities.swift - loads the contents of the university that allows the users to scroll and press on card.\
3) CSUCardview.swift - contains code to display image card following with name. Also tap gesture.\
4) UCCardview.swift - same as previous but with CSU\
5) UCscroll and CSU scroll - contains the code that allows users to scroll through entire csu and uc catalog\
6) starsView.swift - code that loads the number of ratings for the school\
7) admin and admission.swift - code that loads the data of admissions of the schools\
8) csudetail.swift and ucdetails.swift - code that loads all the information contents of schools\
9) sample.swift - contains general code that loads information for csu\'92s.\
